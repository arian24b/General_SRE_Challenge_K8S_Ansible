# Site Reliability Engineer Challenge

## بخش اصلی

### 1. ایجاد سرورها با Terraform
سرورهای لازم را با استفاده از **Terraform** ایجاد کنید (VMها).

### 2. راه‌اندازی کلاستر Kubernetes با Ansible
با استفاده از **Ansible** یک کلاستر Kubernetes راه‌اندازی کنید.
یک کلاستر ساده با یک Node هم کافی است.

### 3. مانیتورینگ کلاستر
مانیتورینگ کلاستر را پیاده‌سازی کنید.
از **Alloy** برای جمع‌آوری داده‌ها استفاده کنید.
برای مشاهده، **Grafana** را راه‌اندازی کنید و **Dashboard** مناسب بسازید.

### 4. Alerting
یک سیستم **Alerting** مناسب پیاده‌سازی شود (برای مشکلات و وضعیت کلاستر).

### 5. راه‌اندازی Postgres روی Kubernetes
یک کلاستر PostgreSQL روی Kubernetes ایجاد کنید.

---

## بخش توسعه اپلیکیشن

### 1. پیاده‌سازی یک Web API
با **Python** یک API بنویسید که:
- یک **IP** را ورودی بگیرد.
- مشخص کند IP مربوط به کدام **کشور** است.

### 2. ذخیره‌سازی داده
نتیجه را روی همان PostgreSQL ذخیره کرده و برای درخواست‌های تکراری از دیتابیس استفاده شود (Cache).

### 3. Deployment روی Kubernetes
اپلیکیشن باید روی کلاستر Kubernetes مستقر شود.

### 4. CI/CD
برای استقرار اپلیکیشن از **Automation** و **CI/CD** استفاده شود.

### 5. Metrics
برای اپلیکیشن **Metric** تعریف کنید و در مانیتورینگ لحاظ کنید.

- برای تمام مراحل **مستندات مناسب** تهیه گردد.
